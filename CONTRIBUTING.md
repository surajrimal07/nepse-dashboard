### Reporting Bugs
- Check the [Issues](https://github.com/surajrimal07/nepse-dashboard/issues) tab to see if the bug has already been reported.
- If not, open a new issue with a clear title and description.
- Include as much info as possible: OS, Browser version, steps to reproduce, and screenshots if applicable.

### Suggesting Enhancements
- Open an issue with the "enhancement" label.
- Explain why this feature would be useful and how it should work.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Install dependencies: `pnpm install`.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes.
5. Make sure your code lints: `pnpm lint`.
6. Issue a pull request!
